# Gmail Sticky Notes

Pin private notes and action items directly to the top of any Gmail thread.

## Features

- Pin multiple notes per email thread
- Reorder notes by dragging or with the keyboard
- Edit notes inline
- Undo accidental deletions
- Badge icons in your email list for threads with notes
- Works in pop-out email windows
- Keyboard accessible throughout

## Privacy

All notes are saved locally using Chrome's `storage.local` API. Nothing is sent to any server, logged, or shared. See [privacy-policy.md](privacy-policy.md).
